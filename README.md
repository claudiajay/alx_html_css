# Responsive Web Page

This project is a responsive web page designed to adapt to different screen sizes, including a mobile layout for screens with a width of 480px or less. 
It includes styling for links and buttons to enhance user experience.

---

## Features

### 1. **Responsive Design**
- Automatically switches to a mobile-friendly layout when the screen width is 480px or less.
- Content is centered on the page with a maximum width of 1000px.

### 2. **Interactive Elements**
- **Links**: Change color to `#FF6565` on hover and when active.
- **Buttons**: Reduce opacity to `0.9` on hover and when active.

---

### Viewing the Web Page
- Resize your browser window to see the layout dynamically adjust for mobile screens.
- Hover over links and buttons to observe the interactive styles.

### Customization
- Modify the `styles.css` file to change colors, fonts, or layouts.
- Adjust breakpoints in the CSS to support additional screen sizes if needed.
